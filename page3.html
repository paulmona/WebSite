How we build this site and host it

Mobirise is an easy static website builder - just drop site elements to your page, add content and style it to look the way you like.

We version control the site using github which integrates to Travis CI.

Travis CI is a cloud based CI/CD tool that seemlessly integrates with Github and deploys the site to  Amazon S3 which hosts the site as a public S3 bucket.

AWS: S3 Bucket in Canada Central Region

Azure: Azure App Services in Central US Region

Google: GCS Bucket in multi EU region.

Finally CloudFlare provides security (DDoS etc) and load balancing across all 3 cloud providers.
