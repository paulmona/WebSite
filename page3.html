What Makes This Site Tick...

How we build this site and host it

Mobirise4

Mobirise is an easy static website builder - just drop site elements to your page, add content and style it to look the way you like.

Github

We version control the site using github which integrates to Travis CI.

Travis CI

Travis CI integrates with Github and deploys the site to  Amazon S3 which hosts the site as a public S3 bucket.

Amazon

We host the siite directly out of Amazon S3 with DNS provided by Amazon Route 53.

Azure

We use Azure App Service to host a geographically redundant and public cloud redundant of the website.  Why?  Because it's easy and doesn't hurt.

Future

Google Compute Platform will be added to round out the top 3.
